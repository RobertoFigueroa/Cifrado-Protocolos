# Cifrado-Protocolos
Implementación básica de protocolos implementado en python

Requerimientos

Librerias requeridas:

    Pyotp 

    Time
    
    pycryptodome
    
    pycryptodomex


La libreria se instala corriendo el siguiente comando:

    Pip install pyotp

    Pip install time

    pip install pycryptodome
    
    pip install pycryptodomex
